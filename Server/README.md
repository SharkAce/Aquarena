# Server
The game server is made using TypeScript in a NodsJS envirement, it handles communication by UDP and TCP sockets.

## Prerequisites
- NodsJS
- tsc

## Installation
```bash
cd Server
npm install
```
or
```bash
cd Server
npm install --include dev
```

## Usage
```bash
npm run start
```

## Compilation
```bash
npm run build-proto
npm run build-ts
```
